Methodology

Best Deal = the knee of the price-vs-intelligence Pareto frontier (nothing cheaper is smarter). Ranked by intelligence gained per 10x of spend, so a cheap-but-weak model can no longer win on the ratio's tail. The Arena view uses the same frontier on Arena quality (arena_coding_elo).

Value Score = Intelligence Index / Blended Price. Higher = more intelligence per dollar (shown in the table; no longer selects the Best Deal).

Deal Score = weighted composite (50% value, 30% coding, 20% speed), each leg log-normalised against fixed reference ranges so scores are comparable across days; value uses a $0.05/1M price floor; models below 95% uptime are excluded from top deals. Used for the table sort and frontier tiebreaks.

Blended Price uses Artificial Analysis's 7:2:1 cache/input/output ratio where available; falls back to 2:1 input/output.

Trends plot raw intelligence and effective blended price over time — never the day-relative deal score, which is not comparable across days. "≥T intelligence" bands are cumulative (all models at or above T).
