# AI Deal Tracker

Daily leaderboard of best-value AI models (intelligence per dollar).

**Live:** [`luarss.github.io/ai-deals`](https://luarss.github.io/ai-deals/)

## Usage

```bash
uv sync
uv run python main.py
open docs/index.html
```

Runs daily at 06:00 UTC via GitHub Actions. Set `DEEPSEEK_API_KEY` as a repo secret for AI insights.
